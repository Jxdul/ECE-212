F.5.5 Saving multiple registers
In an earlier example, we had
main
11 do stuff
BL
subr3
Il calls the subroutine
I1 do more stuff
subr3 PUSH
{R1}
11 save R1
PUSH
{R2}
// save R2
Il do calculations involving R1, R2
POP
{R2}
11 restore R2
POP {R1} I1 restore R1
BX
LR
Il return from subroutine
