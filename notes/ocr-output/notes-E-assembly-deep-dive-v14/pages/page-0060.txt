Branch with link and exchange
BLX{cc} Rm
= Branch to an address held in register Rm, return address saved in LR
• Like BX, allows for change of instruction set, but not in ARMv7-M (since M
profile supports Thumb instruction set only).
• Like BL, also writes the address of the subsequent instruction to LR.
